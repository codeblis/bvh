📖 Descripción

Esta landing page representa la identidad digital de la Bolsa de Valores de La Habana (BVH), un proyecto que rescata el legado financiero cubano y lo proyecta hacia el siglo XXI con una infraestructura moderna, transparente y educativa.

El sitio incluye:

Ticker en tiempo real con datos simulados de empresas cubanas.
Índices sectoriales (General, Tecnología, Turismo, Agroindustria, Energía).
Tabla de cotizaciones con precios y variaciones.
Historia y valores institucionales.
Servicios para emprendedores (educación, consultoría, club de inversión).
Instituto BVH con oferta formativa.
Habana Stock Exchange (HSE) como núcleo operativo.
Formulario de suscripción a boletín.
Modo claro/oscuro con persistencia en localStorage.
Diseño 100% responsivo y accesible.
🚀 Características

✅ Ticker animado con actualización en tiempo real (simulación).

⚙️ Cómo ejecutar localmente

Clona este repositorio:

bash
git clone https://github.com/tu-usuario/bvh-landing.git
Abre el archivo index.html en tu navegador favorito.
(Opcional) Usa un servidor local como Live Server en VS Code para una experiencia más realista.
🌐 Despliegue en GitHub Pages

Sube los archivos a un repositorio en GitHub.
Ve a Settings → Pages.
Selecciona la rama main y la carpeta raíz (/ (root)).
Guarda y espera unos minutos.
Tu sitio estará disponible en https://tu-usuario.github.io/nombre-repo/.
⚠️ Importante: Si el JavaScript no se ejecuta, asegúrate de crear un archivo vacío llamado .nojekyll en la raíz del repositorio para evitar que Jekyll modifique tu código.

🎨 Personalización

Colores: Modifica las variables CSS en :root y body.light-theme dentro de style.css.
Datos del ticker: Edita el array tickerData en el bloque <script> de index.html.
Empresas de la tabla: Actualiza el <tbody> de la tabla .empresas-table.
Cursos del Instituto: Modifica los .course-card en la sección #instituto.
